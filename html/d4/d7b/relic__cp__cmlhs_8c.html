Verifies a CMLHS signature over a set of messages.

Parameters
[in]r- the first component of the homomorphic signature.
[in]s- the second component of the homomorphic signature.
[in]sig- the BLS signatures.
[in]z- the powers of the outputs of the PRF.
[in]a- the vector of first components of the signatures.
[in]c- the vector of second components of the signatures.
[in]msg- the combined message.
[in]data- the dataset identifier.
[in]h- the random element (message has one component).
[in]label- the integer labels.
[in]hs- the hash values, one per label.
[in]f- the linear coefficients in the function.
[in]flen- the number of coefficients.
[in]y- the public elements of the users.
[in]pk- the public keys of the users.
[in]slen- the number of signatures.
[in]bls- the flag for selecting BLS or ECDSA.
Returns
a boolean value indicating the verification result.
