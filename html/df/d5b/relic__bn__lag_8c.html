Computes the coefficients of the polynomial representing the Lagrange interpolation for a modulus and a given set of roots. Computes c(x) = \prod_{0 <= i < n}(x - ai) mod q.

Parameters
[out]c- the coefficients of the polynomial.
[in]a- the set of roots.
[in]b- the modulus.
[in]n- the number of roots to interpolate.
