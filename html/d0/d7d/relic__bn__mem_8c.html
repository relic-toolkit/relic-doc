void bn_init ( bn_t  a,
int  digits 
)

Reduces a multiple precision integer modulo another integer. This macro should not be called directly. Use bn_mod with 4 arguments instead.

Parameters
[out]C- the result.
[in]A- the the multiple precision integer to reduce.
[in]M- the modulus.
[in]U- the auxiliar value derived from the modulus. Reduces a multiple precision integer modulo a positive integer using Montgomery reduction. Computes c = a * u^(-1) (mod m).
[out]C- the result.
[in]A- the multiple precision integer to reduce.
[in]M- the modulus.
[in]U- the reciprocal of the modulus. Exponentiates a multiple precision integer modulo another multiple precision integer. Computes c = a^b mod m. If Montgomery reduction is used, the basis must not be in Montgomery form.
[out]C- the result.
[in]A- the basis.
[in]B- the exponent.
[in]M- the modulus. Computes the greatest common divisor of two multiple precision integers. Computes c = gcd(a, b).
[out]C- the result;
[in]A- the first multiple precision integer.
[in]B- the second multiple precision integer. Computes the extended greatest common divisor of two multiple precision integers. This function can be used to compute multiplicative inverses. Computes c = gcd(a, b) and c = a * d + b * e.
[out]C- the result;
[out]D- the cofactor of the first operand, cannot be NULL.
[out]E- the cofactor of the second operand, can be NULL.
[in]A- the first multiple precision integer.
[in]B- the second multiple precision integer. Generates a probable prime number.
[out]A- the result.
[in]B- the length of the number in bits. Initializes a previously allocated multiple precision integer.
[out]a- the multiple precision integer to initialize.
[in]digits- the required precision in digits.
Exceptions
ERR_NO_MEMORY- if there is no available memory.
ERR_PRECISION- if the required precision cannot be represented by the library.
