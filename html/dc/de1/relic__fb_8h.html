Computes the repeated Frobenius (squaring) or inverse Frobenius (square-root) of a binary field element. If the number of arguments is 3, then simple consecutive squaring/square-root is used. If the number of arguments if 4, then a table-based method is used and the fourth argument is a pointer fo the precomputed table. The variant with 4 arguments should be used when several 2^k/2^-k powers are computed with the same k. Computes c = a^(2^b), where b can be positive or negative.

Parameters
[out]C- the result.
[in]A- the binary field element to exponentiate.
[in]B- the exponent.
[in]...- the modulus and an optional argument.
